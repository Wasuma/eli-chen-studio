Notion for Yoga Instructors & Wellness Professionals: Studio & Class Management System

Master class schedules, student rosters, payments, and wellness programs from a single workspace

TL;DR: Yoga teachers and wellness professionals juggle multiple class schedules, hundreds of students, varying payment models, and wellness program tracking. This guide shows you how to build a Notion system that consolidates student management, tracks attendance and progress, automates payments, and provides personalized wellness tracking—so you can focus on teaching and serving your students.

The Yoga Instructor's Challenge

Whether you're teaching at a studio or running an independent practice, you're dealing with:

  • Scattered student info: Names, emails, phone numbers, preferences scattered across texts, spreadsheets, and Instagram DMs
  • Attendance chaos: No clear view of who's consistently showing up vs. who quit
  • Payment confusion: Mix of class passes, monthly subscriptions, drop-in rates, and barter arrangements
  • Limited visibility into student progress: Hard to track who's improving, who needs modifications, who might need one-on-one attention
  • Personalization at scale: Can't remember individual student injuries, preferences, or goals when teaching large classes
  • Marketing friction: Manually texting or emailing class updates to 100+ students
  • Revenue leakage: Forgotten payments, class passes that expired without being used, unclear billing

Why Notion Solves This for Yoga Professionals

Notion becomes your studio command center because it:

  • Centralizes all student data: One place for names, contact info, preferences, goals, and medical needs
  • Tracks attendance: See who's consistent, who's lapsed, and create win-back campaigns
  • Automates billing: Class passes, subscriptions, and drop-in tracking with payment reminders
  • Supports personalization: Pull up student profiles before class to remember modifications and progress
  • Enables progress tracking: Students can log their practice, goals, and breakthroughs
  • Scales communication: Send class reminders, wellness tips, and updates to segments of students automatically
  • Creates accountability: Students see their attendance, progress, and payment status in a shared view

The Notion System: Core Architecture for Yoga Studios

A production-ready yoga studio workspace contains 6 core databases:

1. Student Roster Database

Complete student information and relationship tracking.

Fields:

  • Student Name: Text
  • Email & Phone: Email and phone number fields
  • Status: Select (Active, Inactive, On Break, VIP)
  • Experience Level: Select (Beginner, Intermediate, Advanced)
  • Medical Notes: Rich text (injuries, disabilities, modifications needed)
  • Personal Goals: Rich text (flexibility, strength, balance, healing, etc.)
  • Preferred Class Times: Multi-select (morning, evening, weekend)
  • Preferred Styles: Multi-select (Vinyasa, Hatha, Restorative, Yin, Hot Yoga, etc.)
  • Referral Source: Select (Friend, Google, Instagram, Studio Walk-in, Other)
  • Emergency Contact: Text (name and phone)
  • Classes Attended: Relation to Class Attendance database
  • Total Classes Attended: Rollup count
  • Last Class Date: Rollup showing most recent attendance
  • Current Payment Status: Status field (Paid, Overdue, Trial, Expired)
  • Notes: Rich text (personality quirks, conversation topics, achievements)

2. Class Schedule Database

All yoga classes offered and their recurring schedules.

Fields:

  • Class Name: Text (e.g., "Monday Morning Vinyasa")
  • Class Style: Select (Vinyasa, Hatha, Restorative, Yin, Kundalini, etc.)
  • Level: Select (All Levels, Beginner-Friendly, Intermediate, Advanced)
  • Instructor: Person field (if multiple instructors)
  • Day of Week: Select (Monday, Tuesday, etc.)
  • Time: Text or time field (6:00 AM, 7:30 PM, etc.)
  • Duration (minutes): Number (45, 60, 75, 90)
  • Location/Studio Room: Text (in-person, online, etc.)
  • Max Students: Number (class capacity)
  • Currently Enrolled: Rollup count
  • Open Spots: Formula (Max - Currently Enrolled)
  • Class Price (Drop-in): Currency
  • Class Roster: Relation to Student database (who's signed up)
  • Next 10 Dates: Rollup or relation to upcoming class instances
  • Class Theme/Focus: Rich text (theme for this month's classes)

3. Class Attendance Tracker

Each actual class session and who attended.

Fields:

  • Date: Calendar field
  • Class: Relation to Class Schedule database
  • Instructor: Person field
  • Attendance List: Multi-select relation to Student database (who showed up)
  • Head Count: Rollup count
  • Notes on Class: Rich text (theme, what went well, adjustments for next time)
  • Guest Instructor Notes: Rich text (guest feedback or observations)

4. Payment & Class Pass Tracker

Billing, class passes, and revenue tracking.

Fields:

  • Student: Relation to Student database
  • Payment Type: Select (Drop-in, Class Pass, Monthly Subscription, Private Session, Retreat)
  • Amount Paid: Currency
  • Date Paid: Date field
  • Class Passes Purchased: Number (5-pack, 10-pack, etc.)
  • Passes Remaining: Number
  • Pass Expiration Date: Date field
  • Status: Status field (Paid, Pending, Overdue, Expired, Refunded)
  • Payment Method: Select (Venmo, PayPal, Cash, Credit Card, etc.)
  • Notes: Rich text (discount codes, promo reasons, etc.)

5. Student Progress & Wellness Tracker

Track student progress, goals, and wellness journey.

Fields:

  • Student: Relation to Student database
  • Entry Date: Date field
  • Practice Type: Select (In-class, Home Practice, Meditation, Other)
  • Duration (minutes): Number
  • How They're Feeling: Select (Energized, Calm, Sore, Struggling, Breakthrough, etc.)
  • Progress Notes: Rich text (improvements noticed, challenges, questions)
  • Photos/Videos: File field (pose progress photos, if student wants to track visually)
  • Related Classes: Relation to classes they took

6. Wellness Resources & Content Library

Reusable yoga sequences, meditation audio, wellness articles, and educational content.

Fields:

  • Content Title: Text
  • Content Type: Select (Yoga Sequence, Meditation, Blog Post, Video, Affirmation)
  • Duration: Text (15 min, 30 min, 45 min, etc.)
  • Level: Select (Beginner, Intermediate, Advanced)
  • Content Focus: Multi-select (Flexibility, Strength, Balance, Stress Relief, Better Sleep, etc.)
  • Link/File: URL or file field
  • Description: Rich text
  • Used in Classes: Relation to Class Schedule (which classes use this sequence)

Pro Workflows: Automations That Save 5+ Hours/Week

Weekly Class Reminders

Use Zapier to send automatic text/email reminders to students the day before their preferred classes:

  • Filter students by preferred class times and styles
  • Auto-send: "Your Monday morning Vinyasa class is tomorrow at 6 AM. See you there!"
  • Track who responds or attends to measure engagement

Attendance Tracking & Win-Back Campaign

Create a filter for students not seen in 4+ weeks:

  • Auto-generate win-back email: "We miss you! Come try our new Friday restorative class this week"
  • Offer specific incentive (free class, discount on pass)
  • Track who returns

Monthly Revenue Report

Notion rollups automatically calculate:

  • Total class revenue (drop-in + passes + subscriptions)
  • Revenue per class
  • New vs. returning student revenue
  • Export for bookkeeping

Class Pass Expiration Alerts

Filter for passes expiring within 7 days:

  • Auto-email students: "Your 5-class pass expires in 7 days! Use your remaining classes or renew your pass"
  • Reduce pass waste and recover revenue

Pre-Class Student Profile Cards

Create filtered views for each class showing:

  • Which students are attending this class
  • Their experience level, goals, and medical notes
  • Modifications they need
  • Personal connection points ("Asked about knee injury last week")
  • Pull these up on phone/tablet before class to personalize teaching

Implementation: 4-Week Rollout

Week 1: Foundation (2-3 hours)

  • Create Student Roster database and add all current students
  • Create Class Schedule database with all your recurring classes

Week 2: Operations (2-3 hours)

  • Create Class Attendance Tracker
  • Log last month of attendance (one-time data entry)
  • Start tracking attendance prospectively

Week 3: Billing (2 hours)

  • Create Payment & Class Pass Tracker
  • Migrate existing payment records
  • Test payment tracking workflow

Week 4: Scaling (2 hours)

  • Add Progress Tracker for students wanting to log practice
  • Create student-accessible view for attendance and progress
  • Set up Zapier automations (reminders, win-back campaigns)

Common Questions

Q: Can my students see and update their own progress in Notion?

A: Yes. Share a read-only/limited-edit view of the Progress Tracker with students. They can log their home practice, but you control what they see (attendance, personal progress, not business data).

Q: How do I handle both online and in-person classes?

A: Create separate entries in Class Schedule or add a "Format" field (In-Person, Online, Hybrid). Attendance tracking works the same way.

Q: Can Notion automatically send payment reminders?

A: Yes, via Zapier. Set up a workflow that checks Payment Tracker for Overdue status and sends email reminders. Cost: $10-20/month for Zapier integration.

Q: How do I track private sessions or yoga therapy separately from group classes?

A: Add a "Session Type" field to Payment Tracker (Group Class, Private Session, Therapy). Create separate enrollment database for private sessions if needed.

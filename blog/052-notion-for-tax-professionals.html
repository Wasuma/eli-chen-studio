The Notion Tax Setup: Core Databases

1. Master Client Database

Centralized client profiles with everything you need:

  • Client name and contact info
  • Entity type: individual, S-corp, C-corp, partnership, LLC, nonprofit
  • Filing status (married, single, head of household)
  • Last year's AGI/income (for context)
  • Estimated tax payment schedule (quarterly vs. annual)
  • Special situations: rental property, business income, investment income
  • Emergency contact (business vs. personal tax contact)
  • Document collection status (full, partial, needs follow-up)
  • Return status: planning, in progress, completed, filed, processed

Pro tip: Add a formula to flag clients where you haven't heard from them in 60+ days. Automatic follow-up reminder.

2. Return Checklist Database

Track what documents you have for each client's return:

  • Client name and return year
  • Document type checklist:
  • ✓ Prior year return
  • ✓ All W-2s
  • ✓ All 1099s (interest, dividends, freelance income)
  • ✓ Mortgage interest statement
  • ✓ Charitable contributions
  • ✓ Medical/dental receipts
  • ✓ Business income records
  • Status: complete, partial, follow-up needed
  • Follow-up date if documents missing
  • Notes on client communication

Filter by status: "Partial — follow up today," "Complete — ready to prepare."

3. Deduction Tracker Database

Help clients track and justify deductions:

  • Client name
  • Deduction category: medical, charitable, business, investment, education
  • Description (e.g., "Medical expenses Q1 2025")
  • Amount claimed
  • Documentation attached (receipts, invoices, bank statements)
  • Status: pending verification, verified, claimed, allowed
  • Notes: any special rules or limitations that apply

Clients appreciate seeing their deductions organized. You have documentation for IRS inquiries.

4. Compliance & Deadline Tracker

Never miss a filing deadline or compliance requirement:

  • Requirement type: individual return, corporate return, 1099 filing, state filing, amendment
  • Client name
  • Due date (federal vs. state, if different)
  • Extension due date (if applicable)
  • Status: not started, in progress, filed, confirmed received
  • Confirmation info: IRS tracking number, state filing number, date received
  • Special notes: any issues or delays

Calendar view shows all upcoming deadlines at a glance. Alerts fire 2 weeks before each deadline.

5. Correspondence Log

Document every client interaction (required for compliance):

  • Date of interaction
  • Client name
  • Type: email, phone call, in-person meeting, IRS notice
  • Subject matter (e.g., "Discussed K-1 from rental property")
  • Summary of what was discussed
  • Any decisions made or actions required
  • Follow-up needed (yes/no, and if yes, due date)

This log protects you if there's ever a dispute about what was said or decided.

6. Tax Planning & Strategy Database

Track quarterly tax projections and planning recommendations:

  • Client name and year
  • Quarterly income projection (Q1, Q2, Q3, Q4)
  • Estimated tax payment amounts (quarterly due dates)
  • Tax-saving strategies identified
  • Recommendations for next year
  • Date of last strategy meeting
  • Follow-up: when to revisit projections

Use this for quarterly check-ins with clients. Proactive tax planning = happier clients = more referrals.

7. Monthly Billing & Time Tracking

Optional but valuable for tracking profitability:

  • Client name
  • Service: tax return prep, planning meeting, IRS response, amendment
  • Hours spent
  • Hourly rate or flat fee
  • Status: invoiced, paid, overdue

See which clients are profitable, which services take the most time, and which clients are slow to pay.

Workflow: Tax Season Management

January: Planning Phase (30 min per client)

  1. Pull up each client profile
  2. Create return checklist for the year (copy template)
  3. Review special situations (rental income, business ownership, etc.)
  4. Set deadline for document collection (e.g., "March 1st")
  5. Send client email with checklist of required documents

February–March: Document Collection (15 min per follow-up)

  1. Check return checklists: which clients are "complete," which are "partial"?
  2. Send reminders to "partial" clients (use template)
  3. Log all correspondence in correspondence database
  4. Update checklist status as documents arrive

April: Return Preparation (2–4 hours per return, depends on complexity)

  1. Open completed checklist for client
  2. Review all attached documents
  3. Prepare return in tax software
  4. Log deductions and strategies in database (for future reference)
  5. Update return status to "completed"

May: Filing & Follow-Up (30 min per return)

  1. Open compliance tracker
  2. File return and update status with confirmation #
  3. Send client copy of filed return + due payment info
  4. Log tax planning recommendations for next year

June–December: Ongoing Support

  1. Quarterly: review tax projections with clients, update estimated payment amounts
  2. Log all client correspondence in correspondence database
  3. Track any IRS notices or amendments needed

Result: Every client is documented, every deadline is tracked, every interaction is logged. If the IRS ever asks, you have everything organized and searchable.
